PyCharm 安装 激活

发表于 2019-07-08 | 分类于 Python

前言 此文只是记录一下 自己在使用PyChram时的 安装 以及激活过程，关于激活 网上有很多文章，本人激活方法是参考 猪哥66 的文章 PyCharm2019 激活码。再此对 猪哥66 表示感谢。

本文 安装的PyCharm 版本是 pycharm-professional-2019.1.3

PyCharm 以及 Python解释器 下载

  • PyCharm 的下载地址：https://www.jetbrains.com/pycharm/download/#section=windows (打开的可能有点慢，稍微等待一下)

    • 打开以后可以看到以下的界面：

    • 此处选择下载 专业版-Professional。

    • 下载好安装包后，进行安装。点击 “Next”

    • 选择安装路径

    • 勾选“create associations（创建关联Python源代码文件）”和“.py”前面的复选框。根据自己需求 勾选是否创建桌面图标(launcher)

    • 点击“Install” 进行自动安装。

    • 自此 PyCharm 安装已经完成了，如果我们之前没有下载有Python解释器的话，我们就要去下载python解释器，不然pycharm只是一副没有灵魂的驱壳。

  • Python 解释器下载地址(如果已有解释器可以直接跳过此处，直接看下面 激活过程):https://www.python.org/

    • 打开以后看到以下界面

    • 点击“Downloads” 选择要下载的版本，此处选择的是 Python 3.7.3

    • 然后选择相关版本 此处选择的是 64位 executable。关于embeddable 稍后说一下

    • 关于 executable 和 embeddable 进行说明一下

      • executable表示可执行版，需要安装后使用。embeddable表示嵌入版，就是解压以后就可以使用的版本。
      • 可执行版安装比较简单，一直默认就好了。embeddable需要注意，当下载好压缩包后是需要解压到同一路径的，这里面放着pip、setuptools等工具，如果不解压，我们将无法在pycharm中更新模块，比如需要用到pymysql，就无法下载。虽然也能用，但是就是”阉割版”的python解释器了。
      • 如果是embeddable版，记得把解释器所在的路径添加到环境变量里，不然pycharm无法自动获得解释器位置。 添加环境变量 此处不再过多叙述。

PyCharm 激活

  • 此处介绍的方法 是使用破解补丁激活

  • 下载破解文件

    • 点击链接 https://pan.baidu.com/s/1R_x0_YGz69mtxC61SJqinw 下载补丁文件 jetbrains-agent.jar 并将它放置到 pycharm安装目录的\bin目录下（位置可随意，放这里是怕误操作删除了破解文件）。

  • 打开PyCharm

    • 如果你是刚下载的pycharm，则需要点击激活窗口的“Evaluate for free”免费试用，然后再创建一个空项目，这样就可以进入到pycharm的工作页面(借图 猪哥66)

    • 然后 点击 创建新项目

    • 点击“Help - Edit Custom VM Options …”修改配置文件,如果提示是否要创建文件，请点”Yes”.

    • 在打开的vmoptions编辑窗口末行添加：-javaagent:你pycharm的安装目录\jetbrains-agent.jar

    • 请仔细检查补丁路径是否正确。

    • 修改完配置文件之后重启Pycharm

    • 修改完配置文件之后重启Pycharm

    • 修改完配置文件之后重启Pycharm

    • 如果错误则会出现Pycharm打不开的情况，这时候可以删除用户配置目录下的pycharm文件夹（能打开pycharm就不要删除）:

    • 输入激活码

      • 选择最后一种License server激活方式，地址填入：http://jetbrains-license-server 完成激活

      • 如果服务器激活方式无法激活，还可以选择Activation code方式激活，复制下面激活码即可

    • 查看有效期

      • 当你激活完毕后，PyCharm右下角会有个Registration小长条提示框，大致的内容为：You copy is Licensed to XXX意思就会告诉你：兄弟，你已经激活成功了，激活码的许可来源是：XXX。

      • 查看有效期的步骤为点击：Help->About这里可以看到你的pycharm的版本号、许可来源、有效期、以及一些环境

        • 服务器激活是没有期限的，即为永久有效

        • 激活码激活的有效期到2089年的

  • 自此 PyCharm pycharm-professional-2019.1.3 激活完毕，以上就是自己安装、激活过程。
